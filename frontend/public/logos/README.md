# Bookmaker Logos

Add the following logo files to this folder:

| File | Bookmaker | Where to get |
|------|-----------|--------------|
| `betway.png` | Betway Ghana | https://www.betway.com.gh - Right-click logo, Save Image |
| `sportybet.png` | SportyBet Ghana | https://www.sportybet.com/gh - Right-click logo, Save Image |
| `1xbet.png` | 1xBet Ghana | https://1xbet.com - Right-click logo, Save Image |
| `22bet.png` | 22Bet Ghana | https://22bet.com - Right-click logo, Save Image |
| `soccabet.png` | SoccaBet Ghana | https://www.soccabet.com - Right-click logo, Save Image |

## Recommended Size
- Width: 120-200px
- Transparent background (PNG) preferred
- Square or horizontal aspect ratio works best

## Fallback
If logo files are missing, the app will display colored badges with initials (BW, SB, 1X, 22, SC).
